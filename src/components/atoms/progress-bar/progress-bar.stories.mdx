## Introduction

A `ProgressBar` should be used to indicate a loading state. This can be useful with view transitions for example.

<CodePreview
  code={`import ProgressBar from "./components/atoms/progress-bar/progress-bar.astro";`}
  label="How to import"
  lang="ts"
/>

</CodePreview>

## Accessibility

When using a `ProgressBar` component, you should provide an `aria-label`.

<CodePreview
  code={`<ProgressBar aria-label="Indeterminate progress" />`}
  label="Improving the accessibility"
  lang="astro"

  <ProgressBar aria-label="Indeterminate progress" />
</CodePreview>
