## Introduction

When using a `CardsList` component, you'll need to provide a function as child that describes how to render the given items. See the example at the end of the [`Astro.slots.render()` section](https://docs.astro.build/en/reference/api-reference/#astroslotsrender) in Astro documentation. If you want to specify the columns, you can use the `--cols` CSS variable.
