## Usage

A <code>CollectionMeta</code> component should be used to display the meta of any collection configured in Astro.

<CollectionMeta data={meta} />

## With hidden label

<CollectionMeta data={meta} hideLabel />

## With icons

<CollectionMeta data={meta} icons={icons} />

## Specific to a collection

The component also handles meta specific to some collections. Check out the following examples.

### Blog post meta

<CollectionMeta data={blogPostMeta} />

### Project meta

<CollectionMeta data={projectMeta} />
