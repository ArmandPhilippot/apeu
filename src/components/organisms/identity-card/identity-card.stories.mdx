## Usage

An <code>IdentityCard</code> component should be used to display information about an author.

## Default

<IdentityCard author={authorPreview} isCentered />

## With avatar

<IdentityCard author={authorPreviewWithAvatar} isCentered />

## With all available data

<IdentityCard author={authorPreviewWithAllInfo} isCentered />

## With heading

<IdentityCard author={authorPreviewWithAllInfo} isCentered>
  <Heading as="h3" slot="heading">About {authorPreview.name}</Heading>
</IdentityCard>
