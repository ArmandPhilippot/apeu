## Introduction

A `CollectionCard` component should be used to display a collection entry preview and to visit a link to learn more about this entry.

<CodePreview
  code={`import CollectionCard from "./components/organisms/collection-card/collection-card.astro";`}
  label="How to import"
  lang="ts"
/>

<CodePreview
  code={`---
const basicCard = {
    collection: "pages",
    description: "A short description of the page.",
    id: "basic-card",
    locale: "en",
    route: "#a-basic-card",
    slug: "a-basic-card",
    title: "The page title",
}
---
<CollectionCard entry={basicCard} headingLvl="h3" />`}
  label="Basic use"
  lang="astro"
>
  <CollectionCard entry={basicCard} headingLvl="h3" />
</CodePreview>
