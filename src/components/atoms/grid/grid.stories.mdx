## Introduction

This component can be used to create a CSS grid.

  lang="ts"
/>

## Options

### `cols`

The `cols` property lets you define an explicit columns number. Caveat: You'll need to handle yourself small viewports.

<CodePreview
  code={`<Grid cols={2}>
    <div>Item 1</div>
    <div>Item 2</div>
    <div>Item 3</div>
    <div>Item 4</div>
    <div>Item 5</div>
    <div>Item 6</div>
</Grid>`}
  label="Setting a number of columns"
  lang="astro"
>
  <Grid cols={2}>
    <div>Item 1</div>
    <div>Item 2</div>
    <div>Item 3</div>
    <div>Item 4</div>
    <div>Item 5</div>
    <div>Item 6</div>
  </Grid>
</CodePreview>

### `rows`

The `rows` property lets you define an explicit rows number. Caveat: This is not useful by itself.

<CodePreview
  code={`<Grid rows={3}>
    <div>Item 1</div>
    <div>Item 2</div>
    <div>Item 3</div>
    <div>Item 4</div>
    <div>Item 5</div>
    <div>Item 6</div>
</Grid>`}
  label="Setting a number of rows"
  lang="astro"
>
  <Grid rows={3}>
    <div>Item 1</div>
    <div>Item 2</div>
    <div>Item 3</div>
    <div>Item 4</div>
    <div>Item 5</div>
    <div>Item 6</div>
  </Grid>
</CodePreview>

### Other CSS grid properties

You can also use `alignContent`, `alignItems`, `justifyContent` and `justifyItems` to customize the alignment of your items in the grid.
