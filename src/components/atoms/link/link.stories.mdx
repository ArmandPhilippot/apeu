## Usage

When used at the beginning or at the end of a sentence, a link should not display an extra space before or after.

<Link href="#destination">A link</Link> at the beginning of a sentence.

A sentence ending with <Link href="#destination">a link</Link>.

## Adding target information

### Language

A link can indicate the language used in the linked resource by using the <code>hreflang</code> attribute.

<p>
  <Link href="#destination" hreflang="fr">A French link</Link>
</p>

### External link

A link can visually indicate that it leads to an external resource either by using the <code>isExternal</code> property or by defining <code>external</code> in the <code>rel</code> attribute.

<p>
  <Link href="#destination" isExternal>An external link</Link>
</p>

<p>
  <Link href="#destination" rel="external">An external link using rel</Link>
</p>

### Download link

A link can visually indicate that it leads to a file (e.g. a PDF) by using the <code>isDownload</code> property.

<p>
  <Link href="#destination" isDownload>A download link</Link>
</p>

### Mixing information

It is also possible to mix the previous information on a same link:

<p>
  <Link href="#destination" hreflang="fr" isExternal>An external French link</Link>
</p>

<p>
  <Link href="#destination" hreflang="fr" isDownload>A French download link</Link>
</p>

<p>
  <Link href="#destination" isDownload isExternal>An external download link</Link>
</p>

<p>
  <Link href="#destination" hreflang="fr" isDownload isExternal>An external download link in French</Link>
</p>

## Wrapping an image

Sometimes links are used to wrap an image (e.g. a gallery). This should also be supported:

<Link href="#destination">
  <img alt="An example" src="https://picsum.photos/100/100" />
</Link>
