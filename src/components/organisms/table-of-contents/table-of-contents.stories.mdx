## Usage

A <code>TableOfContents</code> component should be used on a page to help user navigates through that page using headings.

## Flat structure

<TableOfContents
  headings={[
    { label: "qui velit ratione", slug: "#rerum-velit-dolorem" },
    { label: "dolorem sit omnis", slug: "#quam-fuga-voluptas" },
    { label: "inventore qui suscipit", slug: "#culpa-asperiores-atque" },
  ]}
/>
