## Usage

A <code>CollectionCard</code> component should be used to display a collection entry preview and to visit a link to learn more about this entry.

<CollectionCard entry={basicCard} headingLvl="h3" />

## Variants

### With collection

<CollectionCard
  entry={basicCard}
  headingLvl="h4"
  isCentered
  showCollection
/>

### With featured meta

<CollectionCard
  entry={project}
  featuredMeta={{ key: "kind" }}
  headingLvl="h4"
  isCentered
/>

### With CTA

<CollectionCard
  cta={{ label: "Read more" }}
  entry={basicCard}
  headingLvl="h4"
  isCentered
/>
