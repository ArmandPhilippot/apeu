## With multiple toggles on the page

If you need multiple toggles on the same page for the same setting key, you must provide an `id`.

<CodePreview
  code={`<ThemeSetting
    id="toggle1"
    label="Choose a theme:"
    setting="shiki"
    variant="toggle"
/>
<ThemeSetting
    id="toggle2"
    label="Choose a theme:"
    setting="shiki"
    variant="toggle"
/>
<ThemeSetting
    id="toggle3"
    label="Choose a theme:"
    setting="shiki"
    variant="toggle"
/>`}
  label="Using multiple toggles on a page"
  lang="astro"
>
  <ThemeSetting
    id="toggle1"
    label="Choose a theme:"
    setting="shiki"
    variant="toggle"
  />
  <ThemeSetting
    id="toggle2"
    label="Choose a theme:"
    setting="shiki"
    variant="toggle"
  />
  <ThemeSetting
    id="toggle3"
    label="Choose a theme:"
    setting="shiki"
    variant="toggle"
  />
</CodePreview>

This limitation does not apply to the switch variant.

<CodePreview
  code={`<ThemeSetting label="Enable dark mode?" setting="shiki" variant="switch" />

  label="Using multiple switches on a page"
  lang="astro"
>
  <ThemeSetting label="Enable dark mode?" setting="shiki" variant="switch" />
  <ThemeSetting label="Enable dark mode?" setting="shiki" variant="switch" />
  <ThemeSetting label="Enable dark mode?" setting="shiki" variant="switch" />
</CodePreview>
