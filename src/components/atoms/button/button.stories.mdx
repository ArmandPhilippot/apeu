## Links as button

Sometimes, links can be designed as a button. This component handles this with the `as` property for `secondary`, `discreet` and `neutral` buttons. If you try to use a link with a `primary` button an error will be thrown.

<CodePreview
  code={`<Button as="a" href="#link" kind="secondary">A secondary button</Button>
<br />
<Button as="a" href="#link" kind="discreet">A discreet button</Button>
<br />
<Button as="a" href="#link" kind="neutral">A neutral button</Button>`}
  label="Using a link instead of a button"
  lang="astro"
>
  <Button as="a" href="#link" kind="secondary">A secondary button</Button>
  <br />
  <Button as="a" href="#link" kind="discreet">A discreet button</Button>
  <br />
  <Button as="a" href="#link" kind="neutral">A neutral button</Button>
</CodePreview>

States should behave the same when using a link instead of a button. However, note that a link can't be disabled.
