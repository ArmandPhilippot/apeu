## Usage

A <code>Switch</code> component should be used with boolean settings.

<Switch
  id="usage-example"
  items={[
    { label: "Item 1", value: "item1" },
    { label: "Item 2", value: "item2" },
  ]}
  label="Usage Example"
  value="item1"
/>
