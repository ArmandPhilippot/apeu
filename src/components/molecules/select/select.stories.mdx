## Hidden label

The `hideLabel` property allows you to visually hide the label while keeping it in the DOM to provide an accessible label for screen readers.

<CodePreview
  code={`<Select
    hideLabel
    label="Choose an option"
    options={[
      {label: "Option 1", value: "option-1"},
      {label: "Option 2", value: "option-2"},
      {label: "Option 3", value: "option-3"}
    ]}

## Using an icon

Using the `icon` property, you can add an icon in front of your select. You can also control its size using the `iconSize` property.

<CodePreview
  code={`<Select
    icon="locale"
    label="Choose an option"
    options={[
      {label: "Option 1", value: "option-1"},
      {label: "Option 2", value: "option-2"},
      {label: "Option 3", value: "option-3"}
    ]}
    value="option-2"
  />`}
