## Introduction

A card can be used inside or outside a container. When wrapped in a container, the cover will be placed on the left of the card when the container reaches the breakpoint.

If you don't wrap the Card in a CSS container, its appearance on large screens will be the same as the mobile view. You cannot see the difference here since main is a container.
