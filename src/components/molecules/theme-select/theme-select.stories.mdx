export const components = mdxComponents;
export const themes = {
  auto: "Auto",
  dark: "Dark",
  light: "Light",
};

## Introduction

A `ThemeSelect` component is used to let the user choose the he wants to use for the website.

<CodePreview
  code={`import ThemeSelect from "./components/molecules/theme-select/theme-select.astro";`}
  label="How to import"
  lang="ts"
/>

<CodePreview
  code={`<ThemeSelect current="auto" label="Choose a theme:" themes={themes} />`}
  label="Basic use"
  lang="astro"
>
  <ThemeSelect current="auto" label="Choose a theme:" themes={themes} />
</CodePreview>

## Controlling code blocks theme

By default, the `ThemeSelect` component will fall back on the global theme, but you can also provide `shiki` as value for `theme` to control code blocks theme.

<CodePreview
  code={`<ThemeSelect
    current="auto"
    label="Choose a code block theme:"
    setting="shiki"
    themes={themes}
  />`}
  label="Defining code block themes"
  lang="astro"
>
  <ThemeSelect
    current="auto"
    label="Choose a code block theme:"
    setting="shiki"
    themes={themes}
  />
</CodePreview>
