### Inner shadows

Inner shadows can be used to create a relief effect. Two tokens are available to match the surface colors.

<TokenPreview isInline token={`--color-shadow-inset`}>
<div
  class="token-box"
  style={`background: var(--color-regular-light);padding: var(--spacing-md);`}
>
  <div
    class="token-box"
    style={`background: var(--color-regular);box-shadow: 0 0 0 2px var(--color-shadow-inset);`}
  />
</div>
</TokenPreview>
