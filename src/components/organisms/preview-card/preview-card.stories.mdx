## Introduction

A `PreviewCard` component should be used to display an entry preview with either a link or a call to action to learn more about this entry.

<CodePreview
  code={`import PreviewCard from "./components/organisms/preview-card/preview-card.astro";`}
  label="How to import"
  lang="ts"
/>

<CodePreview
  code={`---
  const basicEntry = {
    description: "The entry excerpt.",
    heading: "The card heading"
  };
  ---
  <PreviewCard entry={basicEntry} headingLvl="h3" />`}
  label="Basic use"
  lang="astro"
>
  <PreviewCard entry={basicEntry} headingLvl="h3" />
</CodePreview>

## Calls to action

### With a linked heading

<CodePreview
  code={`---
  const entryWithLink = {
    description: "The entry excerpt.",
    heading: { label: "The card heading", path: "#a-link" }
  }
  ---
  <PreviewCard entry={entryWithLink} headingLvl="h3" />`}
  label="Wrapping with a link"
  lang="astro"
>
  <PreviewCard entry={{...basicEntry, heading: { label: "The card heading", path: "#a-link" }}} headingLvl="h3" />
</CodePreview>

### With a call to action

<CodePreview
  code={`---
  const entryWithCta = {
    cta: [{label: "Call to action", path: "#call-to-action"}],
    description: "The entry excerpt.",
    heading: "The card heading"
  }
  ---
  <PreviewCard entry={entryWithCta} headingLvl="h3" />`}
  label="Using a call to action"
  lang="astro"
>
  <PreviewCard entry={{...basicEntry, cta: [{label: "Call to action", path: "#call-to-action"}]}} headingLvl="h3" />
</CodePreview>

### With multiple calls to action

<CodePreview
  code={`---
